# Technical Context

## Технологический стек
- **Язык**: Python 3.13
- **Фреймворк**: Flask (для API)
- **HTTP Клиент**: httpx (поддержка асинхронности и стриминга)
- **Аутентификация**: Google OAuth2, Google Service Accounts
- **Управление зависимостями**: uv

## Среда разработки
- **ОС**: Linux (Ubuntu 24.04 LTS)
- **Контейнеризация**: Docker, Docker Compose

## Внешние зависимости
- Google Cloud Vertex AI API
- Google Cloud Cloud Code API (внутренние эндпоинты)
- Google OAuth 2.0
