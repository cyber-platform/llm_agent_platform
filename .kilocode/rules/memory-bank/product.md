# Project Brief: Gemini OpenAI Proxy

## Цель проекта
Создание прокси-сервера, который позволяет использовать модели Google Gemini через API, совместимый с OpenAI. Это необходимо для интеграции Gemini в инструменты, которые изначально поддерживают только OpenAI (например, Kilo Code, Cline и др.).

## Ключевые возможности
- Эмуляция OpenAI Chat Completions API.
- Поддержка стриминга (SSE).
- Поддержка вызова инструментов (Function Calling).
- Два режима работы:
    - **Quota Mode**: Использование OAuth токенов (эмуляция Gemini CLI/Cloud Code) для обхода стандартных лимитов.
    - **Vertex AI Mode**: Использование сервисных аккаунтов Google Cloud Vertex AI.
- Автоматический рефреш токенов в фоновом режиме.
- Маппинг имен моделей и поддержка fallback-цепочек.

## Целевая аудитория
Разработчики, использующие AI-агентов и инструменты автоматизации, желающие использовать мощности Gemini с минимальными изменениями в конфигурации существующих инструментов.
