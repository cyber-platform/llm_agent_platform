# Git Context: Проекты и Репозитории

Этот файл содержит карту всех Git-репозиториев внутри проекта. Используется AI-агентами для правильной навигации и выполнения Git-команд.

## 🗺️ Карта репозиториев

В этом файле перечислены репозитории внутри текущей рабочей области проекта.

| Путь | Описание | Роль |
| :--- | :--- | :--- |
| `./` | Основной проект Gemini OpenAI Proxy | Root Repo |
| `./.kilocode/` | Правила, memory bank и навыки | Nested Repo |
| `./gemini-cli/` | Репозиторий gemini cli (внешний, его код мы не меняем) | Nested Repo |
| `./qwen-code/` | Репозиторий qwen-code (внешний reference upstream, его код мы не меняем) | Nested Repo |
| `./kilocode/` | Репозиторий расширения kilocode для VS code (внешний, его код мы не меняем) | Nested Repo |

## ⚠️ Правила работы

1. **Изоляция**: При внесении изменений в файлы внутри вложенного репозитория, Git-команды (`add`, `commit`, `push`) выполняются внутри соответствующей директории.
2. **Атомарность**: Изменения, затрагивающие несколько репозиториев, фиксируются отдельными коммитами в каждом репозитории.
